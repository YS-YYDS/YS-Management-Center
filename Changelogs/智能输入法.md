# 智能输入法 版本变更日志

---

## [v1.3.0] - 2026-05-07

### 架构升级
- 入口文件模块化拆分：感应引擎、序列化、自启动、GUI 组件独立为模块
- 合并 SetIME 至 YS_Environment，统一 IME 控制入口

### 修复
- 修复「完全停止实例」按钮失效问题，增加三层关闭保障机制
- 修复自启动功能在 cmd_id 为 0 时崩溃的问题
- 修复序列化对混合表（数组+哈希）丢失键的问题
- 修复 ImGui_DestroyContext 缺少 API 存在性检查

### 优化
- 新增跨平台检测，macOS 下给出提示而非静默失败
- 拖拽 Logo 时改为松手后保存配置（防抖）
- UI 文本替换 Unicode 符号为 ASCII

---

## [v1.2.3] - 2026-05-06

### 修复与优化
- 重构底层焦点投递引擎，引入「多目标异步投递」，完美穿透 Windows 10/11 的“不同窗口不同输入法”隔离机制
- 深度性能榨取：彻底移除 defer 循环中的冗余安全校验开销
- 缓存性能升级：废弃字符串转换，直接使用指针映射进行窗口查表，大幅降低 GC 压力
- 匹配算法升级：将所有窗口类名识别引擎从正则匹配改为纯文本匹配，实现扫描速度的质变提升

---

## [v1.2.2] - 2026-05-02

### 修复与优化
- 优化跨版本配置升级的稳定性
- 提升系统级 API 调用兼容性

---

## [v1.2.1] - 2026-04-18

### 修复与优化
- 修复模块路径加载失效及图标资产键名不匹配问题
- 优化 Context 生命周期管理，解决关闭 Logo 后的资源占用与报错
- 实现圆形 Logo 显示与 70% 透明度视觉效果，移除冗余调试输出
- 优化输入法防抖逻辑与焦点切换响应速度
- 正式推出「智能输入法 Lite」版本，支持无 UI 极简工作流

- 优化中英文切换的底层检测逻辑
