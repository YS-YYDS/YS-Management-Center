# 智能输入法 版本变更日志

---

## [v1.2.1] - 2026-04-18

### Bug 修复
- 修复 SetupLoadPath 未添加 Modules 路径导致 require 失败
- 修复 YS_LogoData 键名大小写不匹配（YS_logo vs YS_Logo）
- 修复 GuardInstance 逻辑错误导致的实例互相终止问题
- 修复关闭 Logo 后不断创建 Context 的资源占用问题
- 修复 ImGui_DestroyContext 为 nil 导致的报错

### UI 改进
- 实现圆形 Logo 显示（使用 AddImageRounded）
- 图片半透明效果（70% 透明度）
- 移除所有调试控制台输出

### 代码质量
- 添加 GUI_DISABLED 标志优化 Context 生命周期管理
- 清理冗余调试日志
