# 脚本管理器 版本变更日志

---

## [v0.8.0] - 2026-05-18

### 更新内容
- macOS 兼容：全面修复 macOS 平台的下载、试用申请、激活码验证等功能。
- 下载源切换：默认使用 Gitee 作为主下载源，提升国内下载速度。
- 反爬绕过：添加浏览器 User-Agent 头，解决 Gitee CDN 访问限制。
- 中文路径支持：下载 URL 自动编码中文字符，避免路径解析失败。
- 平台兼容标记：插件卡片新增平台兼容性检测，不支持当前系统的插件会显示灰色蒙层和"不支持"标记。
- 防重复下载：新增下载状态锁，防止用户快速点击导致重复下载任务。
- 调试清理：移除所有调试控制台输出，保持界面整洁。

---

## [v0.7.0] - 2026-05-06

### 更新内容
- UI 体验提升：主界面右下角新增低调的版本号显示，支持鼠标悬停与点击交互。
- 管理器更新日志：现在可以直接点击右下角的版本号，随时查看脚本管理器自身的更新历史。
- 日志阅读优化：大幅优化了更新日志的显示排版，增加了行间距与版本分割线，长篇更新记录也清晰易读。
- 智能解析引擎：底层更新日志解析器全面升级，增强对各类非标准 Markdown 格式的容错与兼容性。
- 安全与稳定性：重构了底层的版本识别机制，隔离了“UI 显示版本”与“底层比对版本”，确保开发者在源码态调试时绝对不会被误触发覆盖安装。
- 核心修复：修复了云端发布清单中因 ID 冲突导致管理器配置被意外覆盖的高危隐患。
