# 拾境 版本变更日志

---

## [v1.3.0] - 2026-05-06
- 钢琴仪表盘：新增双端推子控件（音量/音高/步进），支持独立调节随机范围的上限与下限阈值。
- 交互优化：修复推子交叉干扰及标题栏拖拽联动问题，推子响应由拖拽触发改为按下即跟踪，消除操作死区。
- UI 收藏夹：修复收藏夹显示全部场景而非已收藏场景的问题
- 预览控制：样本管理器预览按钮支持播放/停止状态切换。
- Tooltip：新增全局及钢琴模式独立的 tooltip 隐藏控制。

---

## [v1.2.6] - 2026-05-02
- 核心优化：引入 ToUTF8 缓存、Modal 注册表缓存及分阶段重初始化，显著提升帧率并防止 UI 卡死。
- UI 与修复：修正侧边栏渲染伪影，重构导出目标智能解析逻辑，优化 Docker 模式背景兼容性。
- 系统适配：迁移字体管理器为共享运行时，增强管理中心路径递归查找算法。

---

## [v1.2.5] - 2026-05-02
- 性能加速：重构数据库为异步写入，增加 ToUTF8 LRU 缓存与 Modal 懒加载，彻底消除卡顿。
- 钢琴模式：修复撤销恢复崩溃及步进失效问题，随机抖动上限提升至 2000ms。
- 导入管理：修复导入冲突弹窗，兼容多种场景目录结构，增强库路径有效性校验。

---

## [v1.2.4] - 2026-05-02
- 修复与优化：修复资源库路径无法修改的严重 Bug，全面升级模块加载机制为 `require` 模式。

---

## [v1.2.3] - 2026-04-13
- 稳定性：优化授权验证流程与设备 ID 获取兼容性，改进云端同步机制。

---

## [v1.2.2] - 2026-04-12
- 改进：修复授权状态显示异常，大幅优化 VIP 脚本启动速度。

---

## [v1.1] - 2026-04-11
- UI 增强：重构工业级响应式弹窗系统，支持管理中心后台自动激活。

---

## [v1.0.2] - 2026-03-15
- 数据加密：升级加密协议，提升跨工程调度的安全性。
